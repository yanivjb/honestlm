# honestlm: Guardrails for Teaching Linear Models

Provides small, opinionated wrappers around familiar linear model
workflows. The package keeps base R and ggplot2 interfaces recognizable
while adding guardrails around common interpretation traps, including
coefficient p-values for multi-level categorical predictors, sequential
Type I sums of squares, and plots whose summaries do not match the
fitted linear model.
